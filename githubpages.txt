1. Here are the steps for creating a live link/URL on GitHub pages
  A. Select a Repository
  B. Select the "Settings" tab
  C. Select "Pages" on the left
  D. Under the "Branch" section change the dropdown from "None" to "main" and click "Save"
  E. Live link/URL has now been created
    I. You can refresh the page to see the live link/URL or type it into the browser based off of the following example...
      a. GitHubUsername.GITHUB.IO/RepositoryName
  F. Every time a new change is pushed up for the repository the live link will display that change
